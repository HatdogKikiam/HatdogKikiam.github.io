# ⚖️ CasePoint  
### AI-Driven Legal RAG System for Court Document Analysis

CasePoint is a high-fidelity prototype of an AI-powered legal research system designed to streamline how legal professionals interact with court documents. It leverages Retrieval-Augmented Generation (RAG) principles to provide context-aware, source-cited answers from uploaded legal case files.

---

## 🚀 Overview

CasePoint reimagines legal research by combining document indexing, semantic search, and AI-assisted reasoning into a unified platform.  

Built for environments such as Regional Trial Courts, the system enhances efficiency, reduces manual workload, and ensures traceability of information through source-linked outputs.

---

## ✨ Key Features

### 🔍 AI-Powered Legal Search
- Ask natural language questions about case documents  
- Receive context-aware answers grounded in uploaded sources  
- Source citation and traceability included in every response  

### 📚 Document Library Management
- Centralized repository of legal case files  
- Categorization (Criminal / Non-Criminal)  
- Search, filter, and organize documents efficiently  

### 🧠 Retrieval-Augmented Generation (RAG)
- Combines document retrieval with generative AI  
- Ensures responses are based on actual case content  
- Reduces hallucination through source grounding  

### 📊 Document Insights
- Extract structured insights from legal documents  
- Identify metadata, classifications, and key attributes  
- Visual summaries for quick understanding  

### 👥 Role-Based Access Control
- **Basic User** – View and query documents  
- **Advanced User** – Upload and manage sources  
- **Admin** – Full system control, user management, logs  

### 💬 Interactive Chat Interface
- Clean, modern conversational UI  
- Multi-model support (e.g., GPT-4 variants)  
- Guided prompts and suggestions

## 🎯 Design Principles

- **Usability-first interface** — intuitive navigation and minimal friction  
- **Clarity of information** — structured layouts and readable typography  
- **Traceability** — every AI output is linked to source documents  
- **Scalability** — designed with real-world judicial systems in mind
